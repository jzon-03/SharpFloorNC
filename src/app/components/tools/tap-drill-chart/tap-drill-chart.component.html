school Quick Reference Guide

Thread Engagement

  • 75% - Standard for most applications
  • 65% - Minimum for structural loads
  • 85% - Maximum practical engagement

Thread Types

  • UNC - Unified National Coarse
  • UNF - Unified National Fine
  • Metric - ISO standard threads
  • NPT - National Pipe Thread (tapered)
  • STI - Screw Thread Insert (Helicoil)

STI Insert Notes

  • 100% thread - Provides full thread strength
  • Repair damaged - Restores stripped threads
  • Aluminum upgrade - Adds steel threads to aluminum
  • Installation - Requires STI tap and installation tool

Drill Selection Tips

  • Harder materials - Use larger drill (70% thread)
  • Blind holes - Consider clearance for chips
  • Through holes - Standard 75% engagement
  • Aluminum - Can use smaller drill (80% thread)
